# login

Login to Nx Cloud

## Usage

```shell
nx login
```

Install `nx` globally to invoke the command directly using `nx`, or use `npx nx`, `yarn nx`, or `pnpm nx`.

## Options

### help

Type: `boolean`

Show help

### nxCloudUrl

Type: `string`

The Nx Cloud URL of the instance you are trying to connect to.

### version

Type: `boolean`

Show version number
